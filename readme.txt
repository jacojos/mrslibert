JaCoJo's Anything Service LLC website starter files

Files included:
- index.html -> the main web page
- style.css -> the colors and layout

How to use:
1. Create a new public GitHub repository named YOURUSERNAME.github.io
2. Upload both files to that repository
3. Go to Settings > Pages
4. Under Build and deployment, choose Deploy from a branch
5. Choose main branch and /root
6. Save and wait a minute or two
7. Your site should appear at https://YOURUSERNAME.github.io
